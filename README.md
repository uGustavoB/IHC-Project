**Instale as dependências:**
   - Com npm:
     ```sh
     npm install
     ```
   - Com yarn:
     ```sh
     yarn install
     ```

**Execute o projeto:**
   - Com npm:
     ```sh
     npm run dev
     ```
   - Com yarn:
     ```sh
     yarn dev
     ```

**Acesse no navegador:**
   Abra o navegador e vá para `http://localhost:5173/`.
